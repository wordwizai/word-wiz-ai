UNLIMITED PRACTICE MODE:

Context: This is unlimited practice mode where students can practice as much as they want.

SENTENCE GENERATION:
- Create standalone sentences (not part of a story)
- Each sentence is independent
- Can be on any age-appropriate topic
- Vary topics to keep practice interesting

GUIDELINES:
- Don't reference previous sentences
- Don't create story continuity
- Make each sentence self-contained
- Focus on phoneme practice, not narrative

APPROPRIATE TOPICS:
- Animals and pets
- Daily activities
- Nature and weather
- Simple adventures
- School and learning
- Family and friends

VARIETY:
Try to vary sentence topics so practice doesn't get repetitive.
