## B. Programmatic SEO Logic

### Programmatic Template Structure

**Phonics Skill Pages (Pages 1, 2, 6, 9):**

**Reusable Template Sections:**
1. Hero section with skill name
2. "What are {skill_name}?" definition
3. "Why {skill_name} matter" section
4. "Common struggles with {skill_name}" (UNIQUE per skill)
5. "How to teach {skill_name}: Step-by-step" (UNIQUE steps)
6. "Practice activities for {skill_name}" (UNIQUE activities)
7. "Progress checklist" (UNIQUE milestones)
8. "When to seek additional help" (template)
9. Related resources (links)

**Variables:**
- `skill_name`: "CVC Words", "Consonant Blends", "Short Vowel Sounds", "R-Controlled Vowels"
- `grade_level`: Dynamically mentioned based on skill
- `sound_examples`: 15+ unique examples per skill
- `practice_activities`: 5+ unique activities per skill
- `common_mistakes`: 3-5 unique mistakes per skill
- `example_sentences`: 20+ unique sentences per skill

**Uniqueness Preservation:**
- Each page: ~2,500 words total
- Template sections: ~800 words (32%)
- Unique content: ~1,700 words (68%)
- No duplicate practice examples across pages
- Different teaching approaches per skill
- Skill-specific troubleshooting

**Problem-Solution Pages (Pages 3, 4, 11, 15):**

**Reusable Template Sections:**
1. Parent testimonial/scenario (UNIQUE)
2. "Is this normal?" section (template with variations)
3. "Why this happens" explanation (UNIQUE per problem)
4. "Immediate action steps" (UNIQUE solutions)
5. "Long-term strategies" (UNIQUE approaches)
6. "Red flags to watch" (UNIQUE per problem)
7. "Success stories" section (template)
8. CTA to Word Wiz AI

**Variables:**
- `problem_description`: Specific reading issue
- `age_range`: Varies per problem
- `root_causes`: 3-5 unique causes per problem
- `solutions`: 5-8 unique action steps
- `timeline`: Expected improvement timeframe

## E. Content Uniqueness Strategy

### Anti-Duplication Measures:

1. **Unique Introductions**
   - Each page opens with a unique parent scenario
   - Different hook for each article
   - Varied emotional appeals

2. **Skill-Specific Examples**
   - No overlapping practice sentences
   - Different word lists per skill
   - Unique activity descriptions

3. **Varied Teaching Approaches**
   - CVC page: Tactile/kinesthetic focus
   - Blends page: Auditory discrimination focus
   - Vowels page: Visual/sound association focus
   - R-controlled: Pattern recognition focus

4. **Different Problem Angles**
   - Each problem-solution page addresses distinct issue
   - No overlapping root causes
   - Unique solution sets

5. **Comparison Differentiation**
   - Each comparison has unique feature matrix
   - Different products/approaches
   - Distinct value propositions

## J. Iteration Plan

### When to Optimize (Based on GSC Data):

**High Impressions + Low CTR (>100 impr, <2% CTR):**
- Action: Rewrite title tag and meta description
- Test: More compelling, action-oriented language
- Deadline: 2 weeks after detection

**Low Impressions (<20 after 30 days):**
- Action: Check indexation, add more internal links
- Test: Build topical authority around page
- Deadline: 1 week after detection

**Position 11-20 (Bottom of page 1):**
- Action: Add more comprehensive content, refresh examples
- Test: Increase word count by 20%, add more media
- Deadline: 3 weeks after detection

**No Clicks after 45 days:**
- Action: Full content audit
- Test: May need keyword pivot or content restructure
- Deadline: Review during 60-day analysis
