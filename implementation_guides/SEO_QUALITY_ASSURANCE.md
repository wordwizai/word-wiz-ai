- [ ] Check Google Search Console for crawl errors

- [ ] Monitor for any 404s or redirect issues

**Tools:**
- Google Search Console
- Google Analytics
- PageSpeed Insights

**Red Flags:**
- Page not indexed after 7 days
- Crawl errors reported
- LCP > 4 seconds
- High bounce rate (>80%)

**Action Items if Red Flags:**
- Request indexing in GSC

## ITERATION PROTOCOL

### When Impressions > 100 but CTR < 2%
**Diagnosis:** Title/meta not compelling

**Action Plan:**
1. Analyze top 3 competing pages' titles
2. Rewrite title with:
   - Number/statistic
   - Power word (proven, essential, complete)
   - Year (2025)
   - Benefit-focused
3. A/B test for 2 weeks
4. Keep winning version

**Example:**
- Before: "How to Teach CVC Words to Struggling Readers"
- After: "7 Proven Ways to Teach CVC Words to Struggling Readers (2025 Guide)"

---

### When Position 11-20 for 4+ Weeks
**Diagnosis:** Content not comprehensive enough

**Action Plan:**
1. Analyze top 5 ranking pages
2. Identify content gaps
3. Add 500+ words addressing gaps
4. Add multimedia (images, tables, callouts)
5. Acquire 1-2 internal links from high-authority pages
6. Wait 3 weeks, reassess

---

### When No Impressions After 30 Days
**Diagnosis:** Not indexed or keyword issue

**Action Plan:**
1. Check indexation: `site:wordwizai.com/[url]`
2. If not indexed:
   - Request indexing in GSC
   - Check robots.txt
   - Verify sitemap submission
   - Add more internal links
3. If indexed but no impressions:
   - Keyword may have zero volume
   - Consider keyword pivot
   - Check if cannibalizing another page
